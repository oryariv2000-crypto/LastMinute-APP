# Design System: Last Minute (רגע אחרון)

## Core Principles
- **Efficiency:** Minimal clicks for the business owner.
- **Urgency:** Highlighting discounts and closing times.
- **Sustainability:** Fresh, eco-friendly visual language.

## Colors
- Primary: #2D6A4F (Sustainability Green)
- Secondary: #1B4332 (Dark Forest)
- Accent: #FF7F51 (Sunset Orange - Urgency)
- Background: #F8F9FA (Off-White)
- Text: #212529 (Dark Charcoal)
- Success: #52B788 (Mint Green)
- Error: #D90429 (Action Red)

## Typography
- Font Family: 'Heebo', sans-serif
- Headings: Bold, 24px-32px
- Body: Regular, 16px
- Labels/Captions: Medium, 14px

## Layout & Spacing
- Base Unit: 8px
- Border Radius: 12px (Standard), 50px (Buttons/Pills)
- Card Padding: 16px
- Section Spacing: 24px-32px

## Components
### Buttons
- Primary: Filled #2D6A4F, White text, Pill shape.
- Action: Filled #FF7F51 (for 'Publish' or 'Buy Now').
- Secondary: Outlined #2D6A4F, 2px border.

### Cards
- Background: #FFFFFF
- Shadow: 0px 4px 12px rgba(0, 0, 0, 0.05)
- Radius: 12px

### Inputs
- Border: 1px solid #DEE2E6
- Focus State: 2px solid #2D6A4F
- Radius: 12px
